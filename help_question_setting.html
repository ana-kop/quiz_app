User Guide to Question Setting

Adding new questions

1. In order to create a new question, click on the map in the spot where you would like the question to be located.

2. A form will appear; fill out the form (all fields are required) and click "Save quiz question" button; your question will then be saved to the database. 

IMPORTANT: If you would like to see your newly added question on the map, you need to refresh the page!

3. Once the page has been refreshed, you will be able to see the marker on the map in the spot where you have added the question.

4. Click on the marker if you would like to see how the user will see your question (you will not be able to answer the question in Desktop view)*.



Additional functionality (accessible from the side menu)

  • Click "Show existing questions" -> "All questions" to see all the questions in the database (created by any user). 
    To remove these questions from the view, click "Remove  existing questions" -> "All questions".

  • Click "Show exisiting questions" -> "Added in the last week" to see all the questions added in the last week (created  by any user). 
    To remove these questions from the view, click "Remove existing questions" -> "Added in the last week".

  • Click "5 most difficult questions" to see a list of questions where most wrong answers were given. 

  • Click "Show participation rates" to see a graph of daily participation rates for the past week - for your user, or all users.


* To view the app in the Quiz Answering mode, you need to make the width of the browser window smaller and then refresh the page; alternatively, you can automatically open the Quiz App on your mobile device (as long as your screen size is smaller than 767px)
